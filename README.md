# mystock

启动程序
 python manage.py runserver -H localhost

激活 virtualenvs：
 source venv/bin/activate

退出 virtualenvs：
 deactivate


virtualenvs 配置：
 virtualenv venv



环境安装：
   pip install flask
   #pip install mysql-connector-python
   export PATH=$PATH:/usr/local/mysql/bin
   pip install MySQL-python

pip install -r requirements.txt

  pip install jupyter

   ipython notebook
   jupyter notebook
